API Documentation

Gunakan API kami untuk mengintegrasikan perangkat Anda secara otomatis ke sistem monitoring. Semua request memerlukan autentikasi melalui API Key.

Authentication

Sertakan API Key Anda di dalam header setiap request menggunakan field x-api-key.

x-api-key: ••••••••••••••••••••••••••••••••

Endpoints

POST /api/device
Submit Log Perangkat

Request Body (JSON)

Field Type Description
device_idstringID unik perangkat
statusstringok, error, or warn
files_countnumberJumlah file terenkripsi
versionstringVersi aplikasi device
ephemeral_pubstringPublic key sementara
encrypted_keystringKey yang terenkripsi

Response

200 OK — Success
401 Unauthorized

Example Requests

cURL 
curl -X POST https://${window.location.host}/api/device \
  -H "x-api-key: YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "device_id": "DEV-01",
    "status": "ok"
  }'
JavaScript 
fetch('https://[HOST]/api/device', {
  method: 'POST',
  headers: {
    'x-api-key': 'YOUR_API_KEY',
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({
    device_id: 'DEV-01',
    status: 'ok'
  })
})
Python 
import requests

res = requests.post(
  'https://[HOST]/api/device',
  headers={
    'x-api-key': 'YOUR_API_KEY',
    'Content-Type': 'application/json'
  },
  json={
    'device_id': 'DEV-01',
    'status': 'ok'
  }
)
