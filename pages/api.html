Dokumentasi API

Integrasikan perangkat Anda secara otomatis menggunakan endpoint terenkripsi kami. Semua request wajib menyertakan API Key yang valid.

Autentikasi

Sertakan API Key dalam request header menggunakan field x-api-key.

x-api-key: ••••••••••••••••••••••••••••••••

Endpoints

POST /api/device
Endpoint Perangkat

Request Body (JSON)

Field Type Description
device_idstringID unik perangkat
statusstringok, error, or warn
files_countnumberJumlah file terenkripsi
versionstringVersi aplikasi device
ephemeral_pubstringPublic key sementara
encrypted_keystringKey yang terenkripsi

Response

200 OK — Success
401 Unauthorized

Request Examples

cURL Terminal 
curl -X POST https://[HOST]/api/device \
  -H "x-api-key: YOUR_KEY" \
  -H "Content-Type: json" \
  -d '{"device_id":"DEV-01"}'
JavaScript Fetch 
fetch('https://[HOST]/api/device', {
  method: 'POST',
  headers: {
    'x-api-key': 'KEY',
    'Content-Type': 'json'
  },
  body: JSON.stringify({id:'DEV-01'})
})
